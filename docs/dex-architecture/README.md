# Mobile-First DEX Architecture (Production Grade)

This document set defines a production architecture for a React Native DEX platform designed for millions of users.

## Scope

- AMM swaps
- Route optimization and aggregator support
- MEV-aware execution path
- Wallet-native mobile UX
- Event-driven realtime updates
- Multi-chain extensibility (EVM first, Solana next)
- Future perp/orderbook compatibility
